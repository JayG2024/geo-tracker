USAGE NOTES:
============
- All logos are designed to work in single color
- The tagline "Generative Engine Optimization" can be removed for compact use
- Recommended colors: Blue (#3B82F6), Indigo (#6366F1), or Black (#000000)
- For web use, the SVG components in GeoTestLogo.tsx are recommended
- For print or documents, any of these ASCII versions work well

IMPLEMENTATION:
===============
To use in your app:

import { GeoTestLogo, GeoTestLogoMinimal, GeoTestLogoTech } from './components/GeoTestLogo';

// Default blue logo with tagline
<GeoTestLogo />

// Custom color without tagline
<GeoTestLogoTech color="#6366F1" showTagline={false} />

// Minimal version
<GeoTestLogoMinimal color="#000000" className="h-12" />

To view all variations:
Navigate to: http://localhost:5173/logo
